# About
This is the repository for the Disability Data Initiative (DDI). It contains the R code for the project.

The project website is: https://www.disabilitydatainitiative.org/

The code in this repository is used in our two databases available at:
Disability Statistics – Estimates (DS-E) - https://ds-e.disabilitydatainitiative.org/DS-E/
Disability Statistics – Questionnaire Review (DS-QR) - https://ds-qr.disabilitydatainitiative.org/DS-QR/
